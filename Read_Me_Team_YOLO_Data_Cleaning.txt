# Data cleaning
This code contains data cleaning for airplane customer satisfaction analysis.
The first goal of this project is to help the airplane company understand what factors affect customer satisfcation

## Getting Started


1. Download "train.csv" and "test.csv" from "https://www.kaggle.com/teejmahal20/airline-passenger-satisfaction?select=train.csv"

2. Use Google Colaboratory to open script: Data_Cleaning_by_Team_YOLO.ipynb


### Prerequisites


What things you need to run the software 


1. Access to Google Colaboratory
